# Lineage Builder

> 계보도(스시·일식 쉐프 라인) 시각 편집기 — TAAM 본 앱과 독립적으로 운영되는 SVG 기반 빌더.

## 개요

TAAM (taam-app.vercel.app) 의 계보도 시스템을 처음부터 다시 설계한 독립 프로젝트.
완성도 검증 후 본 앱에 import 또는 JSON 데이터로 통합.

## 핵심 규칙 (LINEAGE_RULES — TAAM 에서 인용)

### 🥇 1순위 — 박스 4면 정중앙 진입
파이프라인의 시작점/끝점은 **항상 노드 박스의 4면 중앙** 중 하나.
박스 안으로 절대 들어가지 않는다. 이름 영역도 침범하지 않는다.

- 상단: `(cx, cy - 29)`
- 하단: `(cx, cy + 29)` (자식 진입) / `(cx, cy + 62)` (부모 출발, 이름 영역 끝)
- 좌측: `(cx - 29, cy)`
- 우측: `(cx + 29, cy)`

### 🥈 2순위 — 화면 위 → 아래 단계적 애니메이션
부모 라인이 노드보다 미리 화면 끝까지 그어지지 않는다.

### 🥉 3순위 — 파이프라인 색상 검정 통일
모든 path stroke = `#1a1a18` (검정).

### 4순위 — 글씨 크기·폰트 통일
- 매장명: `font-family: 'Oswald','Noto Sans KR','Noto Sans JP',sans-serif`, weight 700, `#1a1a18`
- 부제: `font-family: Oswald,sans-serif`, `#999`

## 노드 타입

| 타입 | 설명 | 디자인 |
|---|---|---|
| **메인 노드** | 계보의 시작점 (예: 스기타, 카네사카 본점) | 두꺼운 테두리 + 큰 박스 |
| **제자/직계 노드** | 분기 또는 직계 라인 | 표준 박스 (58×58) |

## 라인 타입

| 타입 | 굵기 | 용도 |
|---|---|---|
| **부모 라인** | 2.5px | 메인 트렁크 / 주요 계보 |
| **가지 라인** | 1.5~1.8px | 분기 / 자식 연결 |

## 라인 편집 기능

- ✅ 가로 / 세로 / 각도 자유 모드
- ✅ 포인트 찍어서 꺾임 추가 (드래그로 위치 조정)
- ✅ 각도 + 길이 수치 입력
- ✅ 박스 4면 정중앙 자동 진입

## 캔버스

- 좌측 타임라인 (연도 세로축, 인라인 편집 가능)
- 세로 길이는 콘텐츠에 따라 자동 확장 / 스크롤

## 기술 스택

- Vanilla JS + 순수 SVG
- 단일 HTML (의존성 0)
- Vercel 정적 배포
- Supabase 연동 (선택, `chefs` 테이블 재사용)

## 로컬 실행

```bash
# 가장 간단한 방법
open builder.html
# 또는
npx serve .
```

## 배포

```bash
vercel --prod
```
